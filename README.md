## 项目说明

本项目是根据慕课网上，@张轩_Viking 老师所开设的一门 `React16 组件化+测试+全流程 实战在线账本项目` 改编而来。
课程地址：
[https://coding.imooc.com/class/302.html](https://coding.imooc.com/class/302.html)

增加了redux、antd等主流的第三方库。

测试方面额外使用了Sinon库。
